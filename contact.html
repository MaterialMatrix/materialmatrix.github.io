Send a request step by step

Work down this list in order. If you do not have every detail yet, send what you have and we can help fill in the gaps.

  1. 01

    Start here

    Open the MaterialMatrix Facebook page

    Project requests are handled through Facebook Messenger. Open the page, click Message, and start a new conversation.

    Open Facebook Page
  2. 02

    Explain the idea

    Tell us what you need made

    A short description is enough to start. Mention what the item is, where it will be used, and what it needs to do.

    Example message I need a replacement bracket for a shelf. It needs to hold light weight, fit in this corner, and be black if possible.
  3. 03

    Add references

    Send photos, measurements, links, or files

    These details help us quote accurately and avoid guessing. Include anything that shows shape, size, fit, or finish.

    Photos or sketches Reference images, broken parts, rough drawings, or screenshots.
    Measurements Width, height, depth, hole sizes, or areas that must fit.
    Files or links STL, STEP, 3MF, model links, or existing design references.
  4. 04

    Set expectations

    Include preferences and timing

    If you have preferences, include them in the first message so we can recommend the right material, color, strength, and print approach.

    Material or strength Decorative, functional, heat-resistant, flexible, or durable.
    Color or finish Preferred color, surface finish, or visible/display requirements.
    Deadline Any event date, pickup target, or timeline that matters.
  5. 05

    Before submitting files

    Confirm you have permission to use the design

    If you send a file or third-party model link, make sure the design allows reproduction or commercial production.
