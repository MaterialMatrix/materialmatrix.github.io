How to send a request

Keep it simple: open the page, message the idea, and include the details that matter most for fit, function, and finish.

01

Open the Facebook page

Click the Facebook button and use the Message option to start the conversation.

02

Describe the project

Tell us what the part or print should do, where it will be used, and any size or fit requirements.

03

Attach references

Add photos, sketches, links, measurements, or existing 3D files so we can review the request accurately.

What to include

Project details

Photos or sketches Reference images, broken part photos, or rough drawings.
Dimensions Width, height, depth, hole sizes, or areas that must fit.
Files STL, STEP, 3MF, or other files if you already have them.

Helpful context

Use and expectations

What it needs to do Decorative, functional, display, replacement, prototype, or gift.
Material needs Strength, heat, flexibility, color, or finish preferences.
Timeline Any deadline, event date, or preferred pickup timing.
